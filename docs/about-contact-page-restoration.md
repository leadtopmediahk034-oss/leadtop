## Scope

- Source pages: `https://www.leadtopmedia.com/aboutus/` and `https://www.leadtopmedia.com/contactus/`
- New routes: `/aboutus` and `/contactus`
- Mode: redesign-preserve. Existing Leadtop navigation, footer, typography, warm-white surface, navy text, orange accent, 8px cards, and pill CTA treatment remain the visual baseline.

## Intentional form change

The legacy Elementor form was replaced with an accessible labeled form. Submission opens the visitor's email client with all fields formatted for `service@leadtopmedia.com`, so the page has a working handoff without introducing an unconfigured backend endpoint.
