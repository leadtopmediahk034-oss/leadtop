- 视觉概念：精密增长仪器。
- 色彩：Helios 纸白、深墨与暖橙体系。
- 字体：Arial、PingFang SC、Microsoft YaHei。
- 形状：8px 圆角与细线校准标记。
- 结构：一张图片只表达一个区块，不以整页长图替代分区效果图。
